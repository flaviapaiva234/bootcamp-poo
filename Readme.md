### Paradigma de Programação Orientado a Objetos

Essa é uma Abstração de um Bootcamp usando Orientação a Objetos em JAVA.

Os 4 pilares principais da Programação Orientada a Objetos são:

- Abstração

- Encapsulamento

- Herança

- Polimorfismo
